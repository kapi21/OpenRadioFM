# Руководство пользователя - OpenRadioFM v3.0 (The Car Experience)

Добро пожаловать в OpenRadioFM v3.0 — финальную эволюцию FM-радио для головных устройств Android (Head Units). Эта версия оптимизирована как для вертикального использования, так и для широкоформатных экранов (1024x600) с новым горизонтальным дизайном.

---

## 1. Интерфейс и навигация

### 1.1 Режимы экрана (Layouts)
OpenRadioFM теперь имеет два основных дизайна:
- **V2 (Вертикальный/Классический):** Оптимизирован для экранов типа планшета или вертикальных дисплеев. *Теперь также поддерживает динамические фоны.*
- **V3 (Горизонтальный/Премиум) [Новинка]:** Широкоформатный дизайн, идеальный для приборной панели, с более крупными значками, логотипами в стиле галереи и графическими индикаторами диапазонов.
- **Как переключить:** Удерживайте кнопку **LOC/DX**, чтобы переключиться между горизонтальным и вертикальным дизайном. Приложение автоматически перезапустится с новым видом.

### 1.2 Нижняя панель управления (V3)
- **Settings (EQ):** Короткое нажатие для эквалайзера. **Длинное нажатие** для меню премиум-персонализации.
- **Band:** Переключение между FM1, FM2 и FM3.
- **LOC/DX:** Изменение чувствительности (локальный/дальний прием). *Длинное нажатие для смены макета.*
- **Scan:** Автоматическое сканирование станций.
- **Mute:** Мгновенное отключение звука.
- **GPS [Новинка]:** Короткое нажатие, чтобы открыть ваше любимое навигационное приложение (Maps, Waze и т. д.).
  - *Скрытое меню:* Сделайте **5 быстрых нажатий** на значок GPS, чтобы открыть заводское техническое меню.

### 1.3 Управление частотой и RDS
- **Левая стрелка (<):** Понижение частоты (-0.05 МГц). Длинное нажатие для автоматического поиска вниз.
- **Правая стрелка (>):** Повышение частоты (+0.05 МГц). Длинное нажатие для автоматического поиска вверх.
- **RDS:** Динамическое отображение названия станции и информационного текста (RadioText) с плавной прокруткой.

## 2. Премиум-персонализация (Секретное меню)

Удерживайте кнопку **Настройки (EQ)** для доступа к центру персонализации:

### 2.1 Цвета темы
Выберите одну из 10 цветовых схем (оранжевый, синий, красный, циан и т. д.), которые применяются к границам и акцентам интерфейса.

### 2.2 Типографика (Шрифты)
Измените вид всех текстов. Доступны:
- **System:** По умолчанию Android.
- **Bebas:** Индустриальный/роботизированный стиль.
- **Digital:** Классический светодиодный стиль старого радио.
- **Inter:** Современный и минималистичный (высокая читаемость).
- **Orbitron:** Футуристический и технологичный стиль.

### 2.3 Режим фона
Управляйте тем, как выглядит фон вашего радио:
1. **Чистый черный:** Идеально подходит для экранов OLED/AMOLED или ночного вождения.
2. **Изображение background.png:** Загрузите свое изображение из `/sdcard/RadioLogos/background.png`.
3. **Динамический логотип [Новинка]:** Фон радио автоматически меняется на логотип настроенной станции с элегантным эффектом размытия (blur).

---

## 3. Управление логотипами и именами

### 3.1 Логотипы станций
- **Расположение:** `/sdcard/RadioLogos/`
- **Вручную:** Вы можете добавить свои логотипы в формате `.png` с названием частоты (например: `96900.png`).
- **Пользовательские имена:** Удерживайте название станции на экране, чтобы переименовать ее. Приложение будет искать логотипы, соответствующие этому имени.

### 3.2 Логотип марки автомобиля [Новинка v3.0]
В горизонтальном макете (V3) вы можете отобразить логотип своего автомобиля с правой стороны:
- **Расположение:** `/sdcard/RadioLogos/car_logo.png`
- **Требование:** Изображение должно называться именно `car_logo.png`. Для лучшей эстетики рекомендуется прозрачный фон.

---

## 4. Устранение неполадок

- **GPS не открывается:** Убедитесь, что у вас установлено навигационное приложение и оно выбрано по умолчанию в Android.
- **Динамический фон не меняется:** Убедитесь, что для настроенной станции назначен логотип. Если логотипа нет, фон станет черным.
- **Логотипы исчезают после перезагрузки:** В версии 3.0 добавлена интеграция со сканером медиафайлов Android, чтобы логотипы всегда сохранялись.
- **Кнопки SEEK инвертированы:** В версии v3.0 ориентация была исправлена, чтобы соответствовать большинству элементов управления на руле.
- **Десятичная точка в частоте:** Теперь принудительно используется точка (например, 108.0) для единообразного отображения во всех системах.
